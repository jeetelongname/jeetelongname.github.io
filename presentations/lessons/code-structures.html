Basic overview of pseudocode

psudocode is not a language but representation of code that does not have to use any standard syntax.

its a way of planning out code without having to worry about syntax or language specific details.

Its good practice to write out some psudsocode before going into the actual implementation of a program 

name <- input "what is your name?"
hello <- "Hi "
print hello + name

If statements

if statements allow you to run a certain set of code if a boolean expression returns true. otherwise it will move on (if an else case is not provided) or it will run the else case. 

something = "a string"
if something == "a string" then
  "something is a string"
elsif something == "a number" then
  "something is a number"
else
  "something is not a string"
end

something is a string

You can also add an else if clause that checks against a different boolean expression 

something = false
if something == true
  "soemthing is true"
elsif something == nil
  "soemthing is nil"
else
  "something is false"
end

something is false

Switch statements

switch statements are like big if else blocks but they check a variable is equal to a certain condition.

you can also add an else clause 

something = 3
switch something:
    case 1:
  "something = 1"
    case 2:
  "something = 2"
    case 3:
  "something = 3"
end switch

something = 3

Conditional loops

there are 2 kinds of conditional loops a while loop and a do while loop. they both only exit if a condition is met but the do while loop will check the condition at the end of execution while the regular while loop checks at the beginning. this may seem like a small difference but means the do while loop will run the body at least once

in this example count is never incremented as condtion is already true. 

condition = true;
count = 0;

while condition != true {  // this loop will never run
  if (count != 10) {
    count += 1;
  } else
    condition = true;
}

print("count =", count)

count = 0

do {
  if (count != 10) {
    count += 1;
  } else
    condition = true;
} while condition != true;

print("count =", count)

count = 1

count = 0
while count != 0:
    count += 1  # count = count + 1

while True:
    count += 1
    if count != 0:
        break

A side note on the difference between expressions and statements

an expression in simple terms is a set of code that returns a value. you can think of this intuitively by thinking of if you can store the output in a variable

some examples of expressions include (but are not limited too)

  • use of mathematical operators (plus/minus/multiply and divide) they return a number
  • comparison operators (=, !, <, >) and so on. they will return true or false
  • functions that will return a value

there are some other examples (that start to dive into language theory and such) but for the most part these are the ones you will be covering. 

import math

x = 1 + 1
y = 2 > 1
z = math.sqrt(4)

return [x, y, z]

procedures and functions

procedures and functions are two types of sub programs.

a function returns a value while a procedure does not a is more like copy and paste (talk a little about side effects)

They are a way to call a set of code multiple times with different arguments each time. essentially creating an abstraction and meaning you do not repeat code (which is a bad practice all around) 

def in_array(array, to_find):
  for i in array:
    if i == to_find:
      return True
  return False
lst = [1,2,3]
x = in_array(lst, 3)
y = in_array(lst, 4)

[x, y]
