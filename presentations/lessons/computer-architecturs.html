The CPU

The CPU is the central core of the computer. it is what processes everything you do. we will look in a little detail how it does that and how it works

The Main Components

ALU

ALU stands for arithmetic logic unit. it is what performs the calculations. the output of which is stored in the accumulator

CU

CU or control unit controls the flow of the CPU it will send out write and read instructions to the memory and is the brain of the brain

Cache

cache is a tiny block of memory right on the CPU. its super fast but super smol used for storing the most used instructions

Registers

Registers are even smaller pieces of memory that are very specialised for specific jobs withing the cpu cycle.

The Fetch Execute cycle

the fetch execute cycle how the cpu runs through the instructions they will fetch the instructions from memory the cpu (control unit) decodes the instruction the cpu executes the instruction we continue until we have no instructions left

Von Neumann

The Von Neuman architecture is the main architecture used by most CPU’s it keeps data and instructions in the same space which makes it much more memory efficient. but means that you need to decode the instructions on the CPU which is less performant
