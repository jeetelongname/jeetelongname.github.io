Why we use what we use

Binary

Computers use binary (other words base 2) as it uses electrical signals for its computation. a 0 is an off state while a 1 is an on state.

Hexadecimal

we use hexadecimal as one digit of hex can represent one nibble or half a byte. this means that the 8 characters used by a single byte can be reduced to two characters. this makes it much more compact making it a convenient short hand

Denary to Binary

to convert from deneray to binary we use a simple table where the first row is the powers of two.

we take the number and work left to right taking away every number that fits so in this case 128, 64, 32 and 16 don’t fit so they get represented with zero’s 8 does fit so we take it away. a one is put in the 8 column 4 does not fit into our new total but 2 does so we put a 1 in the two column we contine until our total is zero

we want to convert 10

  • 10 - 8 = 2
  • 2 - 2 = 0

Binary to Denary

for the reverse we again put our binary number into the table add up all of the powers of 2 which have a one in there column

128 64 32 16 8 4 2 1
0 0 0 0 1 0 1 0

8 + 2 = 10

The basics of Hexadecimal

Hexadecimal is base 16 we can use the same techniqe as before but its easier to just use this trick (both are valid in the exam except one is much faster)

we also need to talk about symbols as hex is base 16 we need some more symbols to represent all the digits so if we wanted to represent 16 it would actually be 10. count up in hex

  • A = 10
  • B = 11
  • C = 12

  • E = 14
  • F = 15

Binary to Hex

for this example we need to first split the byte in half. lets take another example 26 00011010

  • we then convert them to denary (usually in our heads.) and convert it to hex
  • digits

our hex digit is 1A

Hex to Binary

to do the reverse we just reverse the process note I don’t discuss converting from denary to Hex as I would recommend you convert to binary before converting to Hex. You need to be careful as it may seem there is a shortcut but it can be deceptive

we want to convert 1A
