Lists

Lists are what are known as a composite data type. You cannot just have a list. you must have a list of something (unless the list is empty). for example you can have a list of any type.

In python we can mix and match types as well. Though in your spec it may say that they can only have one type (and in many languages which check your types more strongly, you will get an error.)

  • [1, 2, 3]
  • [’a’, ’b’, ’c’]
  • [True, False, True]
  • [1, ’a’, False]

Loops

Its all well and good to have a collection of values, but what if we want to use this collection? maybe print it out?. or more generally, how do we write code that will be run many times?

This is where loops come in. Loops are a statement that allows us to run code many times, either a definite amount of times (for ten iterations as an example) or while a value remains true.

For loops

There are 2 kinds of for loop you need to know about. The count controled loop and the collection loop

Count Controlled

Lets say we want to iterate 10 times we can use the range construct. This gives us a range we can move over. 

for i in range(10):
    print(i)

Adding in some more details

We can also add in an optional start value and a step, like so 

for i in range(start := 2, stop := 12, step := 2):
    print(i)

While loops

While loops are simpler, they just need a boolean expression These loops can run for an indefinite amount of time, because the boolean expression may never become false 

a = 0
while a != 10:
    print(a)
    a += 1

a = 0
while a != 10:
    print(a)
