Breaking flow

there are two keywords that allow you to break or continue loops while its going. they are called break and continue 

for elem in [1,2,3]:
    if elem == 2:
        break
    else:
        print(elem)

1

for elem in [1,2,3]:
    if elem == 2:
        continue
    else:
        print(elem)

Return values

to return a value you have to use the return keyword, this stops the flow of execution within the function 

def return_value(arg):
    print("hey!")
    return arg
    print("I will never be reached!")


print(return_value(1))
